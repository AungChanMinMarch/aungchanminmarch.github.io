Suppose $\limit f_n(x) = f(x) \ (x\in E)$.Put $M_n=\sup\limits_{x\in E} \mod{f_n(x)-f(x)}$.

Then $f_n \to f$ uniformly on $E$ if and only if $M_n \to 0$ as $n\to \infty$

Suppose $f_n \to f$ uniformly on $E$.

Let $\varepsilon > 0$.

$$\begin{align*} \mbox{Then there is an integer } &N \mbox{ such that }\\ n \geq N, x \in E &\mbox{ implies } \abs{f_n(x) - f(x)} \leq \ep\\ n \geq N &\mbox{ implies } \sup\limits_{x\in E} \abs{f_n(x) - f(x)} \leq \ep\\ n \geq N &\mbox{ implies } M_n \leq \ep \\ n \geq N &\mbox{ implies } \abs{M_n - 0} \leq \ep \end{align*}$$

Hence $M_n \to 0$ as $n \to \infty$.

Conversely, suppose $M_n \to 0$ as $n\to \infty$.

Then for every $\varepsilon > 0$, there exists an integer $N$ such that

$$\begin{align*} %\mbox{Then for every } \varepsilon > 0, \mbox{there exists an integer } &N \mbox{ such that }\\ n &\geq N &&\mbox{ implies that }& \abs{M_n - 0} &\leq \varepsilon \\ n &\geq N& &\mbox{ implies that }& M_n &\leq \varepsilon \\ n &\geq N& &\mbox{ implies that }& \sup_{x\in E} \abs{f_n(x) - f(x)} &\leq \varepsilon (\forall x \in E) \\ %n \geq N,& x \in E& &\mbox{ implies that }& \sup_{x\in E} \abs{f_n(x) - f(x)} &\leq \varepsilon \\ n \geq N,& x\in E& &\mbox{ implies that }& \abs{f_n(x) - f(x)} &\leq \varepsilon \\ \end{align*}$$

Hence $f_n \to f$ uniformly.
