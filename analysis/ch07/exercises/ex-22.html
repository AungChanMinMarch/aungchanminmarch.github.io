Assume $f \in \fR(\alpha) \mathcal{R}$ on $[a,b]$, and prove that there are polynomials $P_n$ such that $$\limit \int_a^b \abs{f-P_n}^2 dx = 0 $$

We want to show that for every $\ep >0$ there exists some polynomial $P(x)$ such that $\bracket{\int\limits_{a}^{b} \abs{f-P}^2 dx}^{\frac{1}{2}} < \ep$

Since $f \in \fR(\alpha)$, there is a continuous function $g$ on $[a,b]$ such that $\| f-g \|_2 < \ep$

Since $g$ is continuous, by 7.26, there is a polynomial $P$ such that $\abs{P(x) -g(x)} < \frac{\ep}{2\sqrt{\alpha(b)-\alpha(a)}}$

Then $$\|f-P\|_2^2 = \int_a^b \abs{g(x) -P(x)}^2 d\alpha < \frac{\ep}{2\sqrt{\alpha(b)-\alpha(a)}} (\alpha(b)-\alpha(a)) = \frac{\ep^2$}{4}$$

Hence $\|g-P\|_2 = \frac{\ep}{2}$

Since $f,g, P \in \fR(\alpha)$, $$\begin{align*} \|f-P\|_2 &\leq \|f-g\|_2 + \|g-P\|_2 \\ &< \frac{\ep}{2} + \frac{\ep}{2} = \ep \\ \end{align*}$$
