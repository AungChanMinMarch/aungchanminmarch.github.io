Assume $f \in \riemannR(\alpha)$ on $[a,b]$, and prove that there are polynomials $P_n$ such that $$\limit \int_a^b \abs{f-P_n}^2 d\alpha = 0 $$

We want to show that for every $\ep >0$ there exists some polynomial $P(x)$ such that $\bracket{\int\limits_{a}^{b} \abs{f-P}^2 d\alpha}^{\frac{1}{2}} = \|f-P\|_2 < \ep$

Since $f \in \riemannR(\alpha)$, there is a continuous function $g$ on $[a,b]$ such that $\| f-g \|_2 < \frac{\ep}{2}$

Since $g$ is continuous, by 7.26, there is a polynomial $P$ such that $\abs{P(x) -g(x)} < \frac{\ep}{2\sqrt{\alpha(b)-\alpha(a)}}$

$$\begin{align*} \mbox{Then } \|g-P\|_2^2 &= \int_a^b \abs{g(x) -P(x)}^2 d\alpha\\ &= \abs{g(x) -P(x)}^2 (\alpha(b)-\alpha(a)) \\ &< \bracket{\frac{\ep}{2\sqrt{\alpha(b)-\alpha(a)}}}^2 (\alpha(b)-\alpha(a)) = \frac{\ep^2}{4}\\ \mbox{Hence } \|g-P\|_2^2 &< \frac{\ep^2}{4}\\ \|g-P\|_2 &< \frac{\ep}{2} \end{align*}$$ $$\begin{align*} \mbox{Since } f,g,P &\in \riemannR(\alpha), \\ \|f-P\|_2 &\leq \|f-g\|_2 + \|g-P\|_2 \\ &< \frac{\ep}{2} + \frac{\ep}{2} = \ep \\ \|f-P\|_2 &< \ep \\ \bracket{\int\limits_{a}^{b} \abs{f-P}^2 d\alpha}^{\frac{1}{2}} &< \ep \\ \int\limits_{a}^{b} \abs{f-P}^2 d\alpha &< \ep^2 \\ \abs{\int\limits_{a}^{b} \abs{f-P}^2 d\alpha - 0} &< \ep^2 \\ \limit \int\limits_{a}^{b} \abs{f-P}^2 d\alpha &= 0\\ \end{align*}$$
