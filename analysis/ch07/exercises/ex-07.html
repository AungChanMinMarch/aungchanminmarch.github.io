For $n=1,2,3,\dots$ and $x$ real, put $$f_n(x)=\frac{x}{1+nx^2}$$

Show that sequence $f_n$ is converges uniformly to a function $f$, and that the equation $$f'(x)= \limit f_n'(x)$$ is correct if $x\neq 0$ and false if $x=0$

$$\begin{align*} (1-\sqrt{n}\mod{x})^2 &\geq 0 \\ 1 - 2n\mod{x} + nx^2 &\geq 0 \\ 1+ nx^2 &\geq 2n \mod{x} \end{align*}$$

By Schwarz inequility, $$\mod{f_n(x)} = \mod{\frac{x}{1+nx^2}} \leq \frac{\mod{x}}{1+nx^2} \leq \frac{\mod{x}}{2\sqrt{n}\mod{x}} = \frac{1}{2\sqrt{n}} \to 0, \text{ for } x\neq 0$$

If $x=0$, then $f_n(0) = 0$ for all $n$

Hence $fn \to 0$ uniformly.

Since the limit function if $f(x)=0$ for all $x \in \R$, $f'(x) = 0$ for all $x$

$$\begin{align*} f_n'(x) &= \frac{(1+nx^2) - x(2nx)}{(1+nx^2)^2}\\ &= \frac{1-nx^2}{1+2nx^2+n^2x^4} \\ \limit f_n'(x) &= \limit \frac{1-nx^2}{1+2nx^2+n^2x^4} \\ &= \limit \frac{\frac{1}{n^2} - \frac{x^2}{n}}{\frac{1}{n^2} + \frac{2x^2}{n}+ x^4} = 0 \text{ for } x\neq 0 \end{align*}$$

Therefore $f'(x)= \limit f_n'(x)$ if $x\neq 0$.

If $x=0$, $f'(0)=1$ and hence $\limit f_n(0) = 1 \neq f'(0)$, which is $0$.
