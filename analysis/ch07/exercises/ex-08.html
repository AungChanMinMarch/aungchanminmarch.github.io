If $$I_n = \begin{cases} 0 &\mbox{if $x\leq 0$} \\ 1 &\mbox{ if $x >0$} \end{cases}$$

if $x_n$ is a sequence of distince points of $(a,b)$ and if $\sum \mod{c_n}$ converges, prove that the series $$f(x) = \summ c_n I(x-x_n) \ (a \leq x \leq b)$$ converges uniformly and that $f$ is continuous for $x \neq 0$

Let $f_n(x) = \sum\limits_{k=1}^{n} c_n I(x-x_n), \ (a \leq x \leq b)$

$$\begin{align*} \text{Then if } n > m,\\ \mod{f_n(x) - f_m(x)} & = \mod{ \sum_{k=1}^{n} c_k I(x-x_k) - \sum_{k=1}^{m} c_k I(x-x_k)}\\ &= \mod{\sum_{k=m+1}^{n} c_k I(x-x_k)}\\ &\leq \sum_{k=m+1}^{n} \mod{c_k I(x-x_k)}\\ &\leq \sum_{k=m+1}^{n} \mod{c_k}\\ \end{align*}$$

Since $\sum \mod{c_n}$ converges, by 3.22, for any $\varepsilon >0$ there is an integer $N$ such that for $n,m \geq N$ such that $$\mod{f_n(x) - f_m(x)} \leq \sum_{k=m+1}^{n} \mod{c_k} < \varepsilon $$ Hence by 7.8, the series converges uniformly.

Let $x \in (a,b)$ such that $x \neq x_n$ for any $n$.

Then the partial sum $f_n$ is constant in any neighborhood of $x$ not containing any of $x_1, x_2, \dots, x_n$.
Hence by 7.11,

$$\begin{align*} \lim_{t \to x} \limit f_n(t) &= \limit \lim_{t \to x} f_n(t) \\ \lim_{t \to x} f(t) &= \limit f_n(t) = f(x) \end{align*}$$

Thus $f$ is continuous at $x$
