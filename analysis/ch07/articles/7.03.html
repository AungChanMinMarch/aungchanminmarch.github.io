Let $f_n: \R \to \R$ be defined by $f_n(x) =\dfrac{x^2}{(1+x^2)^n}, (n=1,2,3,...)$
  1. Does $\{f_n\}$ converge pointwise?
  2. Are the functions $f_n$ and the limit function $f$ continuous?
  3. Is the sum of the series $\Sum_{n=0}^{\infty}f_n$ continouos?
  1. \text{For } x \neq 0, & \Lim_{n \to \infty} f_n(x) &= \limit \dfrac{x^2}{(1+x^2)^n} = 0 \\ \text{For } x = 0, & \Lim_{n \to \infty} f_n(0) &= \limit 0 = 0

    Since $\{f_n(x)\}$ converges every $x\in \R$, $\{f_n\}$ converges pointwise.

  2. Each $f_n$ is continuous since it is rational function.

    Since the limit function $f(x)=\limit f_n(x)=0$ for all $x \in \R$, it is continuous.

  3. \text{For } x \neq 0,& \summ f_n(x) &=\summ \dfrac{x^2}{(1+x^2)^n} \\ &&=x^2 \summ \dfrac{1}{(1+x^2)^n}\\ &&=x^2 (1+\dfrac{1}{1+x^2} + \dfrac{1}{(1+x^2)^2} + \dfrac{1}{(1+x^2)^3}+...)\\ &&=x^2 \dfrac{1}{1-\dfrac{1}{1+x^2}}\\ &&=x^2 \dfrac{1+x^2}{1+x^2-1}\\ &&=1+x^2 \\ \text{For } x = 0,& f_n(0) &= 0 \\ & \summ f_n(0) &= 0

    Define $f(x)=\Sum_{n=0}^{\infty} f_n(x)$ by $f(x)= \begin{cases} 0 & \mbox{ if $x=0$}\\ 1+x^2 & \mbox{ if $x\neq 0$} \end{cases}$

    The sum of the series is not continuous at $x=0$.
