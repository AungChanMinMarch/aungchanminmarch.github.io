For $m=1,2,3,\dots,$ put $f_m(x)=\Lim_{n\to \infty} (\cos m!\pi x)^{2n}$.
Show that each function $f_m$ and the limit function of $\{f_m\}$ are discontinous.

For $m=1,2,3,...$, $f_m(x)=\Lim_{n\to \infty} (\cos m!\pi x)^{2n}$

When $m!x$ is an integer, say $a$, $$\begin{align*}(\cos m!\pi x)^{2n} = (\cos a \pi)^{2n} = \{(-1)^a\}^{2n} = (-1)^{2na}=1 \\ f_m(x) = \limit (\cos m!\pi x)^{2n} = \limit 1 = 1\end{align*} $$

For all other values of $x$,$$\begin{align*} |\cos m! \pi x | < 1 \\ |\cos m! \pi x |^2 < 1 \\ f_m(x) = \limit (\cos m! \pi x)^{2n} = 0 \end{align*}$$

Hence $f_m(x) = \begin{cases} 1 &\mbox{if $m!x$ is an integer} \\ 0 &\mbox{if $m!x$ is not integer} \end{cases}$

Therefore each $f_m$ is discontinuous.

Let the limit function $f(x)=\Lim_{m \to \infty} f_m(x)$

If $x$ is irrational, then $m!x$ can never be integer. Hence $f_m(x) =0$

If $x$ is rational, say $\dfrac{p}{q}$, then $m!x= m!\dfrac{p}{q}$ is an integer if $m \geq q$. ($m$ will be greater than $q$ as $m \to \infty$) Hence $\Lim_{m \to \infty} f_m(x) = 1$

Therefore, $f(x) = \Lim_{m \to \infty} f_m(x) = \begin{cases} 0 \mbox{ if $x$ is irrational} \\ 1 \mbox{ if $x$ is rational} \end{cases}$

Therefore $f$ is discontinuous.
