Pointwise Convergenet

Suppose $\{f_n\}, n=1,2,3...$ is a sequence of functions defined on a set $E$ and suppose that the sequence of numbers $\{f_n(x)\}$ converges for every $x\in E$. We can then define a function $f$ by $$f(x)=\lim_{n \to \infty}f_n(x) \ \ \ (x \in E)$$

Then $\{f_n\}$ converges to $f$ pointwise on $E$ and $f$ is called the limit or limit function of $\{f_n\}$

$f_n(x)=nx$ for all $x \ \in \ R$
$$\begin{align*} \text{For all } x \ &\in R \\ \lim_{n \to \infty}f_n(x) &= \lim_{n \to \infty} nx \\ &= \infty \end{align*}$$

Since $f_n(x)$ does not converge, $\{f_n\}$ is not pointwise convergent.

$f_n(x)=\frac{nx+x^2}{n^2}$ for all $x \ \in \ R$
$$\begin{align*} \text{For all } x \ &\in R \\ \lim_{n \to \infty}f_n(x) &= \lim_{n \to \infty} \frac{nx+x^2}{n^2} \\ &= \lim_{n \to \infty} \frac{x}{n}+\frac{x^2}{n^2} \\ &= 0 + 0 \\\ &= 0 \end{align*}$$

$\{f_n\}$ converges pointwise.

$f_n(x)=\frac{\sin(nx+3)}{\sqrt{n+1}}$ for all $x \in R$

For every $x \in R$, we have

-1 & \leq & \sin(nx+3) & \leq & 1 \\ \frac{-1}{\sqrt{n+1}} & \leq & \frac{\sin(nx+3)}{\sqrt{n+1}} & \leq & \frac{1}{\sqrt{n+1}} \\ \Lim_{n \to \infty}\frac{-1}{\sqrt{n+1}} & \leq &\Lim_{n \to \infty} \frac{\sin(nx+3)}{\sqrt{n+1}} & \leq & \Lim_{n \to \infty}\frac{1}{\sqrt{n+1}} \\ 0 & \leq & \Lim_{n \to \infty} f_n(x) & \leq & 0

Since $\Lim_{n\to \infty}f_n(x) = 0$ for all $x$ in $R$, $\{f_n\}$ converges pointwise.

$f(x)=n^2x^n$ for $0 \leq x \leq 1$s
$$\begin{align*} &\text{For } x=0, & f_n(0) &= 0 \\ &\text{For } 0 < x < 1, & \Lim_{n\to \infty}f_n(x) &= \Lim_{n\to \infty} n^2 x^n \\ &&&= \Lim_{n \to \infty} \frac{n^2}{x^{-n}} (\frac{\infty}{\infty}) \\ &&&= \Lim_{n \to \infty} \frac{2n}{-x^{-n}} (\frac{\infty}{\infty}) \\ &&&= \Lim_{n \to \infty} \frac{2}{x^{-n}} \\ &&&= \Lim_{n \to \infty} x^n \\ &&&= 0 (\because 0 < x < 1)\\ &\text{For } x=1, & f_n(1) &= n^2 \\ && \Lim_{n\to \infty} f_n(1) &= \Lim_{n \to \infty} n^2 = \infty \end{align*}$$

Therefore $\{f_n\}$ does not pointwise converge

A Convergent series of continuous functions may have a discontinuous sum.

Let $f_n: \R \to \R$ be defined by $f_n(x) =\frac{x^2}{(1+x^2)^n}, (n=1,2,3,...)$
  1. Does $\{f_n\} converge pointwise?
  2. Are the functions $f_n$ and the limit function $f$ continuous?
  3. Is the sum of the series $\Sum_{n=0}^{\infty}f_n$ continouos?
  1. \text{For } x \neq 0, & \Lim_{n \to \infty} f_n(x) &= \limit \frac{x^2}{(1+x^2)^n} = 0 \\ \text{For } x = 0, & \Lim_{n \to \infty} f_n(0) &= \limit 0 = 0

    Since $\{f_n(x)\}$ converges every $x\in \R$, $\{f_n\}$ converges pointwise.

  2. Each $f_n$ is continuous since it is rational function.

    The limit function $f(x)=\limit f_n(x)$ is continuous since $f(x)=0$

  3. \text{For } x \neq 0,& \summ f_n(x) &=\summ \frac{x^2}{(1+x^2)^n} \\ &&=x^2 \summ \frac{1}{(1+x^2)n}\\ &&=x^2 (1+\frac{1}{1+x^2} + \frac{1}{(1+x^2)^2} + \frac{1}{(1+x^2)^3}+...)\\ &&=x^2 \frac{1}{1-\frac{1}{1+x^2}}\\ &&=x^2 \frac{1+x^2}{1+x^2-1}\\ &&=1+x^2 \\ \text{For } x = 0,& f_n(0) &= 0 \\ & \summ f_n(0) &= 0

    Define $f(x)=\sum_{n=0}^{\infty} f_n(x)$ by $f(x)= \begin{cases} 0 & \mbox{ if $x=0$}\\ 1+x^2 & \mbox{ if $x\neq 0$} \end{cases}$

    The sum of the series is not continuous.

For $m=1,2,3,...,$ put $f_m(x)=\Lim_{n\to \infty} (\cos m!\pi x)^{2n}$.
Show that each function $f_m$ and the limit function of $\{f_m\}$ are discontinous.

For $m=1,2,3,...$, $f_m(x)=\Lim_{n\to \infty} (\cos m!\pi x)^{2n}$

When $m!x$ is an integer, say $a$, $$\begin{align*}(\cos m!\pi x)^{2n} = (\cos a x)^{2n} = \{(-1)^a\}^{2n} = (-1)^{2na}=1 \\ f_m(x) = \limit (\cos m!\pi x)^{2n} = \limit 1 = 1\end{align*} $$

For all other values of $x$,$$\begin{align*} |\cos m! \pi x | < 1 \\ f_m(x) = \limit (\cos m! \pi x)^2n = 0 \end{align*}$$

Hence $f_m(x) = \begin{cases} 1 &\mbox{if $m!x$ is an integer} \\ 0 &\mbox{if $m!x$ is not integer} \end{cases}$

Therefore each $f_m$ is discontinuous.
