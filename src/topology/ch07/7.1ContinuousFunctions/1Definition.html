--- title: "Definition: Continuous Function" order: 1 ---
Let $(X, \topologyTau)$ and $(Y, \topologyTau^*)$ be topological spaces. A function $f$ from $X$ into $Y$ is continuous relative to $\topologyTau$ and $\topologyTau^*$, or $\topologyTau-\topologyTau^*$ continuous, or simply continuous, iff the inverse image $f^{-1}[H]$ of every $\topologyTau^*$-open subset $H$ of $Y$ is a $\topologyTau$-open subset of X, that is, iff $$H \in \topologyTau^* \text{ implies } f^{-1}[H] \in \topologyTau$$

    Hence $f$ is continuous for any $\topologyTau$.

Proposition 7.1 (Eg 7.3)

Prove that a function $f: X \to Y$ is continuous iff the inverse of each member of a base $\base$ for $Y$ is an open subset of $X$.

Suppose the inverse of each member of a base $\base$ for $Y$ is an open subset of $X$.

Let $H$ be an open subset of $Y$.
Since $\base$ is a base for the topology on $Y$, $H$ is a union of members of base $\base$.
\begin{align} H&=\cup_{i} B_{i}, \text{ where } B_i \in \base \\ f^{-1}[H] &=f^{-1}\left[\cup_{i} B_{i}\right] =\cup_{i} f^{-1}\left[B_{i}\right] \end{align} Since $f^{-1}[B]$ is an open subset of $X$ for each member $B \in \base$, each $f^{-1}\left[B_{i}\right]$ is open.
Hence $f^{-1}[H]$ is the union of open sets and hence $f^{-1}[H]$ is open. Thus $f$ is continuous.

Now suppose $f$ is continuous.

Let $\base$ be a base for a topology of $Y$.

Then $\base$ is a class of open sets of $Y$.

Hence if $B \in \base$, $B$ is an open set of $Y$. Since $f$ is continuous, $f^{-1}[B]$ is open

Therefore $f^{-1}[B] is open for every set $B$ in $\base$

Therorem 7.2 (= Ex no.6)

Let $\subbase$ be a subbase for a topological space $Y$. Then show that a function $f: X \to Y$ is continuous iff the inverse of each member of $\subbase$ is an open subset of $X$.

Let $\subbase$ be a subbase for a topological space $Y$.

Then $\subbase$ is a class of open sets of $Y$, i.e. every set in $\subbase$ is an open set of $Y$.

Suppose a function $f: X \to Y$ is continuous.

Then the inverse image of every open set of $Y$ is an open set of $X$.

Hence the inverse image of every set in $\subbase$ is an open set of $X$

Suppose the inverse of each member of $\subbase$ is an open subset of $X$.

Let $H$ be an open set of $Y$.

Since $\subbase$ is a subbase for a topological space $Y$, $H = \cup_i \bracket{ S_{i_1} \cap \cdots \cap S{i_{n_i}} }$, where $S_{i_k} \in \subbase$.

\begin{align} H &= \cup_i \bracket{ S_{i_1} \cap \cdots \cap S_{i_{n_i}} }, \text{ where } S_{i_k} \in \subbase \\ f^{-1}[H] &=f^{-1} \left[ \cup_i \bracket{ S_{i_1} \cap \cdots \cap S_{i_{n_i}} } \right] \\ &= \cup_i \bracket{ f^{-1}[S_{i_1}] \cap \cdots \cap f^{-1}[S_{i_{n_i}}] } \end{align}

Since the inverse of each member of $\subbase$ is an open subset of $X$, each $f^{-1}[S_{i_k}]$ is open.

Thus $f^{-1}[H]$ is the union of finite intersection of open set and hence $f^{-1}[H]$ is open.

Therefore $f$ is continuous.

Ex no. 1

Prove: Let $f:X \to Y$ be a constant function, say $f(x)=p \in Y$, for every $x \in X$. Then f is continuous relative to any topology $\topologyTau$ on $X$ and any topology $\topologyTau^*$ on $Y$.

We need to show that the inverse image of any $\topologyTau^*$-open subset of $Y$ is a $\topologyTau$-open subset of $X$.

Let $H \in \topologyTau^*$.

Since $f(x) = p$ for all $x \in X$, $$ f^{-1}[H] = \begin{cases} X &\mbox{ if } p \in H \\ \emptyset &\mbox{ if } p \notin H \end{cases}$$

since $X$ and $\emptyset$ belong to every topology $\topologyTau$ on $X$, $f^{-1}[H]$ is an open subset of $X$.

Thus $f$ is continuous

Eg 1.5

Show that the absolute value function $f$ on $\R$, ie. $f(x) =|x|$ for $x\in \R$, is continuous.

Let $A = (a,b)$. $$f^{-1}[A] = \begin{cases} \emptyset &\mbox{ if } a < b \leq 0 \\ (-b, b) &\mbox{ if } a < 0, b > 0 \\ (-b, -a) \cup (a,b) &\mbox{ if } 0 \leq a < b \\ \end{cases}$$

In each case, $f^{-1}[A]$ is open. Hence $f$ is continuous.

Ex no 3

Let $\mathscr{U}$ be the usual topology on the real line $\R$ and let $\topologyTau$ be the upper limit topology on $\R$ which is generated by the open-closed intervals $(a, b]$. Furthermore, let $f: \R \to \R$ be defined by $$f(x) = \begin{cases} x &\mbox{ if } x \leq 1 \\ x+2 &\mbox{ if } x > 1 \end{cases}$$

  1. Show that $f$ is not $\usual$-$\usual$ continuous.
  2. Show that f is $\topologyTau$-$\topologyTau$ continuous.

  1. $(-1, 2)$ is $\usual$-open set of $\R$ but $f^{-1}[(-1, 2)] = (-1, 1] $ not $\usual$-open set of $\R$. Thus $f$ is not $\usual$-$\usual$ continuous.
  2. Since $\topologyTau$ is the upper limit topology on $\R$ which is generated by the open-closed intervals $(a, b]$, $\topologyTau$ is a class of all open-closed intervals $(a,b]$. Let $A = (a,b]$. $$f^{-1}[A] = \begin{cases} (a, b] &\mbox{ if } a < b \leq 1 \\ (a, 1] &\mbox{ if } a < 1, 1 < b \leq 3 \\ (a, b-2] &\mbox{ if } a < 1, b > 3 \\ \emptyset &\mbox{ if } 1 \leq a < b \leq 3 \\ (1, b-2] &\mbox{ if } 1 \leq a, 3 < b \\ (a-2, b-2] &\mbox{ if } 3 \leq a < b \\ \end{cases}$$

    In each case, $f^{-1}[A]$ is a $\topologyTau$-open set. Hence $f$ is $\topologyTau$-$\topologyTau$ continuous.

Ex no 4

Suppose a function $f:(X,\Tau_1) \to (Y,\Tau_2)$ is not $\Tau_1$-$\Tau_2$ continuous. Show that if $\Tau_1^*$ is a topology on $X$ coarser than $\Tau_1$, and if $\Tau_2^*$ is a topology on $Y$ finer than $\Tau_2^*$, then $f$ is also not $\Tau_1^*$-$\Tau_2^*$ continuous.

Since a function $f:(X,\Tau_1) \to (Y,\Tau_2)$ is not $\Tau_1$-$\Tau_2$ continuous, there exist a $H \in \Tau_2$ such that $f^{-1}[H] \notin \Tau_1$.

Since $\Tau_1^*$ is a topology on $X$ coarser than $\Tau_1$, we have $\Tau_1^* \subset \Tau_1$.

$$ \therefore f^{-1}[H] \notin \Tau_1 \text{ implies } f^{-1}[H] \notin \Tau_1^*$$

Since $\Tau_2^*$ is a topology on $X$ finer than $\Tau_1$, we have $\Tau_2^* \supset \Tau_2$, i.e. $\Tau_2 \subset \Tau_2^*$.

$$ \therefore H \in \Tau_2 \text{ implies } H \in \Tau_2^*$$

Thus we have $H \in \Tau_2^*$ but $f^{-1}[H] \notin \Tau_1^*$ and hence $f$ is not $\Tau_1^*$-$\Tau_2^*$ continuous.

Ex no 5

Show that the identity function $i: (X,\Tau) \to (X,\Tau^*)$ is continuous if and only if $\Tau$ is finer than $\Tau^*$

By definition, $i$ is $\Tau$-$\Tau^*$ continuous if and only if

$$ H \in \Tau^* \text{ implies } i^{-1}[H] \in \Tau$$

Since $i$ is the identity function, $i^{-1}[H] = H$. Thus we get

\begin{align} i \text{ is continuous } &\iff H \in \Tau^* \text{ implies } H \in \Tau \\ i \text{ is continuous } &\iff \Tau^* \subset \Tau \\ i \text{ is continuous } &\iff \Tau \text{ is finer than} \Tau^* \end{align}

ex 8

Prove: Let the functions $f: X \to Y$ and $g:Y \to Z$ be continuous. Then the composition function $g \composite f: X \to Z$ is also continuous.

Let $H$ be an open subset of $Z$.

Since $g:Y \to Z$ is continuous, $g^{-1}[H]$ is an open subset of $Y$.

Since $f:X \to Y$ is continuous, $f^{-1}[g^{-1}[H]]$ is an open subset of $X$.

Since $(g \composite f)^{-1}[H] = f^{-1}[g^{-1}[H]]$, $(g \composite f)^{-1}[H]$ is an open subset of $X$.

Thus the composition function $g \composite f: X \to Z$ is also continuous

ex 9

Prove: Let $\{\Tau_i\}$ be a collection of topologies on a set $X$. If a function $f:X \to Y$ is continuous with respect to each $\Tau_i$, then $f$ is continuous with respect to the intersection topology $\Tau = \cap_i \Tau_i$

Let $H$ be an open subset of $Y$.

a function $f$ is continuous with respect to each $\Tau_i$, $f^{-1}[H]$ belong to each $\Tau_i$ and hence $f^{-1}[H] \in \cap_i \Tau_i$.

Thus $f$ is continuous with respect to $\cap_i \Tau_i$.

ex 10 = Th 7.3

Prove Theorem 7.3: A function $f:X \to Y$ is continuous if and only if the inverse image of every closed subset of $Y$ is a closed subset of X.

Suppose $f:X \to Y$ is continuous.

Let $F$ a closed subset of $Y$.

Then $F^C$ is an open subset of $Y$.

Since $f:X \to Y$ is continuous, $f^{-1}[F^C]$ is an open subset of $X$.

Since $f^{-1}[F^C] =(f^{-1}[F])^C$, $(f^{-1}[F])^C$ is an open subset of $X$.

Thus $f^{-1}[F]$ is a closed subset of $X$.

Suppose the inverse image of every closed subset of $Y$ is a closed subset of X.

Let $H$ be an open subset of $Y$.

Then $H^C$ is a closed subset of $Y$.

Since the inverse image of every closed subset of $Y$ is a closed subset of X, $f^{-1}[H^C]$ is a closed subset of $X$.

Since $f^{-1}[H^C] = (f^{-1}[H])^C$, $(f^{-1}[H])^C$ is a closed subset of $X$.

Thus $f^{-1}[H]$ is an open subset of $X$ and hence $f$ is continuous.
