OPEN AND CLOSED FUNCTIONS

  1. A function $f: X \to Y$ is called an open (or interior) function if the image of every open set is open.
  2. A function $g: X \to Y$ is called a closed function if the image of every closed set is closed.

Eg 2.1

Consider the projection mapping $\pi: \R^2 \to \R$ of the plane $\R^2$ into the x-axis, i.e. $\pi((x,y)) = x$. Show that $\pi$ is open but not closed.

Suppose that $G$ is an open subset of $\R^2$. We need to prove $\pi[G}$ is open.

Observe that the projection $\pi[D]$ of any open disc $D \subset \R$ is an open interval.

Let $\pi(p) \in \pi[G]$. Then $\pi(p)$ belongs to an open interval contained in $\pi[G]$, i.e. $\pi(p) \in (\pi(p) - \ep, \pi(p) +\ep) \subset \pi[G]$ for some $\ep > 0$. Hence $p$ is an interior point of $\pi[G]$ and $\pi[G]$ is open.

$\pi$ is open.

Let $A = \{(x, y)| xy \geq 1, x>0 \}.

Clearly, $A$ is a closed subset of $\R^2$.

$\pi[A] = (0, \infinity)$ is not closed.

Hence $\pi$ is not closed.

Bicontinuous Function

If $f$ is open and continuous, $f$ is called bicontinuous or topological.

HOMEOMORPHIC SPACES

Two topological spaces $X$ and $Y$ are called homeomorphic or topologically equivalent if there exists a bijective (i.e. one-one, onto) function $f:X \to Y$ such that $f$ and $f^{-1}$ are continuous. The function $f$ is called a homeomorphism.

$f: X \to Y$ is a homeomorphism iff $f$ is bicontinuous and bijective.
