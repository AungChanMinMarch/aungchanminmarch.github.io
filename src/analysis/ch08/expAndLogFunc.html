THE EXPONENTIAL AND LOGARITHMIC FUNCTIONS

$$\begin{align} \text{We define } E(z) &= \summ \frac{z^n}{n!} \\ \text{Then } E(z) &= \frac{z^0}{0!} + \frac{z}{1!} + \frac{z^2}{2!} + \frac{z^3}{3!} + \dots \\ &= 1 + z + \frac{z^2}{2!} + \frac{z^3}{3!} + \dots \\ \abs{\frac{a_{n+1}}{a_n}} &= \abs{\frac{z^{n+1}}{(n+1)!} \times \frac{n!}{z^n}} \\ &= \frac{z}{n+1} \end{align}$$

We consider the sequence $\{\frac{\abs{z}}{n+1}\}$, i.e. $\abs{z}, \frac{\abs{z}}{2}, \frac{\abs{z}}{3}, \dots$. The limit of each subsequence of this sequence is $0$. Thus $\limit \sup \abs{\frac{a_{n+1}}{a_n}} = 0$.

That is, $\limit \sup \abs{\dfrac{\frac{z^{n+1}}{(n+1)!}}{\frac{z^n}{n!}}} = 0 <1$. By root test, this series converges.
