--- title: Theorem 3.39 parent: rudinAnalysis03 ---
Given the power series $\sum c_n z^n$, put $\alpha = \limit \sup \sqrt[n]{\abs{c_n}}, R = \frac{1}{\alpha}$. If $\alpha= 0, R = + \infty$; if $\alpha = + \infty, R=0$. Then $\sum c_nz^n$ converges if $\abs{z} < R$ and diverges if $\abs{z} > R$.
