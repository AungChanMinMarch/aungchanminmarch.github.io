Let $\{f_n\}$ be a sequence of continuous functions which converges uniformly to a function $f$ on a set $E$. Prove that $$\limit f_n(x_n) = f(x)$$ for every sequence of points $x_n \in E$ such that $x_n \to x$ and $x\in E$. Is the converse of this true?

Let $\ep >0$

Since $\{f_n\}$ converges uniformly to $f$, there is an integer $N_1$ such that
$n \geq N$ and $y \in E$ implies that $\abs{f_n(y) - f(y)} < \frac{\ep}{2}$

Since $\{f_n\}$ be a sequence of continuous functions which converges uniformly to a function $f$ on a set $E$, by 7.12 $f$ is continuous on $E$.

Hence there is $\delta >0$ such that
$$\abs{x-y} < \delta \text{ implies } \abs{f(x)-f(y)} < \frac{\ep}{2} \tag{1}$$

Since $x_n \to x$, there is an interger $N_2$ such that
$$n \geq N_2 \text{ implies } x-x_n < \delta \\$$ Therefore we get $\abs{f(x) -f(x_n) < \frac{\ep}{2}}$ from (1)

Hence if $n \geq \max(N_1, N_2)$, $$\begin{align*} \abs{f_n(x_n) -f(x)} &= \abs{f_n(x_n)-f(x_n) +f(x_n) -f(x)}\\ &\leq \abs{f_n(x_n) -f(x_n)} + \abs{f(x_n)-f(x)}\\ & < \frac{\ep}{2} + \frac{\ep}{2} = \ep\\ \therefore \limit f_n(x_n) &= f(x) \end{align*}$$

The converse is not true. For example, Let $E=[0,1)$ and $f_n(x) =x^n$.

Then the sequence converges to the constant function $f(x)=0$ but not uniformly.

If $\{x_n\}$ is a sequence of points of $E$ convergeing to a point $x$ then it is contained in a closed subinterval $[0,a]$ where $a < 1$

Since $\{f_n\}$ converges uniformly on this subinterval, $\{f_n(x_n)\}$ converges to $f(x)$.

Hence the property is insufficient for determining that the convergence is uniform.
