Suppose $\{f_n\}$ is an equicontinuous sequence of functions on a compact set $K$, and $\{f_n\}$ converges pointwise on $K$. Prove that $\{f_n\}$ converges uniformly on $K$.

Let $\ep > 0$.

Since $\{f_n\}$ is equicontinuous on $K$, there exists $\delta >0$ such that $$x,y \in K, d(x,y) < \delta, n\in \N \text{ implies } \mod{f_n(x) - f_n(y)} < \dfrac{\ep}{3}$$

Let $N_{\delta}(x) = \{y \mid y\in K, d(x,y) < \delta\}$

Since $K$ is compact, there are finitely many points $x_1, x_2, \dots, x_m$ in $K$ such that

$$K \subset N_{\delta}(x_1) \cup N_{\delta}(x_2) \cup \dots \cup N_{\delta}(x_m)$$ $$\begin{align*} \text{Hence if } y \in K, \text{ then } y &\in N_{\delta}(x_i) \text{ for some }i = 1,2,3,...,m \\ d(x_i,y) &< \delta \\ \mod{f_n(x_i) - f_n(y)} &< \dfrac{\ep}{3} \\ \therefore y \in K \text{ implies } \mod{f_n(x_i) - f_n(y)} &< \dfrac{\ep}{3} \text{ for all $n$ and for some }i = 1,2,3,...,m \end{align*}$$

Since $\{f_n\}$ converges pointwise on $K$, there exists an integer $N_x >0$ such that $$n,m > N_x \text{ implies } \mod{f_n(x)-f_m(x)} < \dfrac{\ep}{3}$$

Put $N = \max(N_{x_1}, N_{x_2}, ..., N_{x_m})$. Then $n,m > N$ implies that

$$\begin{align*} n, m &> N_{x_i} \text{ where }i = 1,2,3,...,m \\ \mod{f_n(x_i)-f_m(x_i)} &< \dfrac{\ep}{3} \text{ for all $n$ and }i = 1,2,3,...,m \end{align*}$$ $$\begin{align*} \therefore y \in K, n,m > N \text{ implies } \mod{f_n(y) - f_m(y)} &\leq \mod{f_n(y) - f_n(x_i)} + \mod{f_n(x_i) - f_m(x_i)} + \mod{f_m(x_i) - f_m(y)} \\ &< \dfrac{\ep}{3} + \dfrac{\ep}{3} + \dfrac{\ep}{3} = \ep \end{align*}$$ Hence by {% a 7.8 %} $\{f_n\}$ converges uniformly on $K$
