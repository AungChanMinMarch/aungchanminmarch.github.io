--- title: "Theorem 7.17" parent: uniformConvergenceAndDifferentiation order: 17 ---
Suppose $\{f_n\}$ is a sequence of functions, differentiable on $[a,b]$ and such that $\{ f_n(x_0) \}$ converges for some point $x_0$ on $[a,b]$. If $\{f_n'\}$ converges uniformly on $[a,b]$, then $\{f_n\}$ converges uniformly on $[a,b]$, to a function $f$, and $f'(x) = \limit f_n'(x) (a\leq x \leq b)$.

Let $\ep >0$.

Since $\{ f_n(x_0) \}$ converges and $\{f_n'\}$ converges uniformly on $[a,b]$, there is an integer $N$ such that

$$n \geq N, m \geq N \mbox{ implies } \abs{f_n(x_0) - f_m(x_0)} \leq \frac{\ep}{2}$$ $$\mbox{and } n \geq N, m \geq N, t \in [a,b] \mbox{ implies } \abs{f_n'(t) - f_m'(t)} \leq \frac{\ep}{2(b-a)} \tag{1}$$

Applying mean value theorem to function $f_n-f_m$ on $a\leq x < t \leq b$,

$$\begin{align*} (f_n-f_m)'(r) &= \frac{(f_n-f_m)(x) -(f_n-f_m)(t)}{x-t}, \mbox{ where $r \in [x,t]$} \\ f_n'(r) - f_m'(r) &= \frac{f_n(x)-f_m(x) - f_n(t)+f_m(t)}{x-t} \end{align*}$$ $$\begin{align*} \mbox{(1) gives } \abs{\frac{f_n(x)-f_m(x) - f_n(t)+f_m(t)}{x-t}} &\leq \frac{\ep}{2(b-a)} \tag{2}\\ \abs{f_n(x)-f_m(x) - f_n(t)+f_m(t)} &\leq \frac{\ep\abs{x-t}}{2(b-a)} \leq \frac{\ep\abs{b-a}}{2(b-a)} = \frac{\ep}{2}\\ \end{align*}$$ $$\begin{align*} \mbox{Hence } \abs{f_n(x) -f_m(x)} &= \abs{f_n(x)-f_m(x) - f_n(x_0)+f_m(x_0) + f_n(x_0) - f_m(x_0)} \\ &\leq \abs{f_n(x)-f_m(x) - f_n(x_0)+f_m(x_0)} + \abs{f_n(x_0) - f_m(x_0)} \\ &\leq \frac{\ep}{2} + \frac{\ep}{2} = \ep \end{align*}$$

Hence by 7.8, $\{f_n\}$ converges uniformly.

Let $f(x) = \limit f_n(x)$.

Fix a point $x$ on $[a,b]$ and put $\phi(t) = \frac{f(t)-f(x)}{t-x}$ and $\phi_n(t) = \frac{f_n(t)-f_n(x)}{t-x}$, where $t \in [a,b]$ and $t \neq x$.

$$\mbox{Then } \Lim_{t \to x} \phi(t) = \Lim_{t \to x} \frac{f(t)-f(x)}{t-x} = f'(x) \tag{3}$$ $$\mbox{Also } \Lim_{t \to x} \phi_n(t) = \Lim_{t \to x} \frac{f_n(t)-f_n(x)}{t-x} = f_n'(x) \text{ for } n=1,2,3, \dots \tag{4}$$ $$\begin{align*} \abs{\phi_n(t) - \phi_m(t)} &= \abs{\frac{f_n(t)-f_n(x)}{t-x} - \frac{f_m(t)-f_m(x)}{t-x}} \\ &= \abs{\frac{f_n(t)-f_n(x) - f_m(t) + f_m(x)}{t-x}} \\ \mbox{(2) gives } \abs{\phi_n(t) - \phi_m(t)}&\leq \frac{\ep}{2(b-a)} \\ \end{align*}$$ Hence by 7.8, $\{\phi_n\}$ converges uniformly for $t\neq x$. $$\begin{align*} \limit \phi_n(t) &= \limit \frac{f_n(t)-f_n(x)}{t-x} \\ &= \frac{\limit f_n(t)- \limit f_n(x)}{t-x} \\ &= \frac{f(t)-f(x)}{t-x} \\ &= \phi(t) \mbox{ uniformly for $t \in [a,b]$ and $t \neq x$.} \end{align*}$$

By 7.11, we get

$$\begin{align*} \Lim_{t\to x} \limit \phi_n(t) &= \limit \Lim_{t\to x} \phi_n(t)\\ \mbox{(4) gives } \Lim_{t \to x} \phi(t) &= \limit f_n'(x)\\ \mbox{(3) gives } f'(x) &= \limit f_n'(x)\\ \end{align*}$$
