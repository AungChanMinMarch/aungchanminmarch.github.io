If $\{f_n\}$ is a sequence of continuous functions on $E$, and if $f_n \to f$ uniformly on $E$, then $f$ is continuous on $E$.

Let $\ep > 0$ and $x_0 \in E$.

Since each $f_n$ is continuous on $E$, it is continuous at $x_0$ in $E$.

Then there exists $\delta > 0$ such that

$$x \in E, \abs{x-x_0} < \delta \mbox{ implies that } \abs{f_n(x) - f_n(x_0)} < \frac{\ep}{3}$$

Since $f_n \to f$ uniformly on $E$, there exists an integer $N$ such that

$$x \in E, n \geq N \mbox{ implies that } \abs{f_n(x) - f(x)} < \frac{\ep}{3}$$

Since $x_0 \in E$, then $n \geq N$ implies that $\abs{f_n(x_0) - f(x_0)} < \frac{\ep}{3}$

Thus $x \in E, \abs{x-x_0} < \delta$ implies that

$$\begin{align*} \abs{f(x) - f(x_0)} &= \abs{f(x) - f_n(x) + f_n(x) - f_n(x_0) + f_n(x_0) - f(x_0)}, \mbox{where $n\geq N$}\\ &\leq \abs{f(x) - f_n(x)} + \abs{f_n(x) - f_n(x_0)} + \abs{f_n(x_0) - f(x_0)}\\ &< \frac{\ep}{3} + \frac{\ep}{3} + \frac{\ep}{3} = \ep \end{align*}$$

Since $x \in E$ and $\abs{x-x_0} < \delta$ implies that $\abs{f(x)-f(x_0)} < \ep$, $f$ is continuous at every $x_0$ in $E$. Thus $f$ is continuous on $E$.

Let $f_n:[0,1] \to \R$ be defined by $f_n(x)=n^2x(1-x^2)^n, (n=1,2,3,\dots)$. Show that each function $f_n$ is continous on $[0,1]$ and the limit function $f$ is continuous on $[0,1]$ but $\{f_n\}$ does not converge uniformly to $f$.

Each $f_n$ is continuous on $[0,1]$ since it is a polynomial function.

$$\begin{align*} \text{If }x=0,\ &f_n(0)=0 \text{ for all } n\\ \text{If }x=1,\ &f_n(1)=0 \text{ for all } n\\ \text{If }0 < x < 1,\ & 0 < x^2 < 1\\ &0 > -x^2 > -1 \\ &1 > 1-x^2 > 0\\ &0 < 1-x^2 <1 \\ & \frac{1}{1-x^2} > 1 \\ \limit n^2x(1-x^2)^n &= x\limit \frac{n^2}{(\frac{1}{1-x^2})^n} \bracket{\text{indeterminate form } \frac{\infty}{\infty}}\\ &= x\limit \frac{2n}{log(\frac{1}{1-x^2})(\frac{1}{1-x^2})^n} \bracket{\text{indeterminate form } \frac{\infty}{\infty}}\\ &= x \limit \frac{2}{\bracket{log(\frac{1}{1-x^2})}^2(\frac{1}{1-x^2})^n}\\ &= \frac{2x}{\bracket{log(\frac{1}{1-x^2})}^2}\limit(1-x^2)^n \\ &= 0 \ (\because 1-x^2 < 1)\\ \therefore \limit f_n(x) &= 0 \forall x \in [0,1] \end{align*}$$

Hence the limit function is $f(x)= \limit f_n(x) = 0$ for every x in $[0,1]$. The limit function $f$ is continuous since it is constant function.

Since $\abs{f_n(x) - f(x)} = \abs{n^2x(1-x^2)^n - 0} = n^2x(1-x^2)^n \nrightarrow 0$, $\{ f_n \}$ does not converge uniformly to $f$.
